## Getting Started

1. **Installation**

   ```bash
   git clone https://github.com/your-org/vert-stack-template my-project
   cd my-project
   pnpm install
   ```

2. **Development**
   ```bash
   pnpm dev        # Start development server
   pnpm test       # Run tests
   pnpm build      # Production build
   ```
